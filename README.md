# 🐹 Learning Go: Exercícios por Etapa

Caderno de prática para aprender Go (Golang) do básico ao avançado.
Cada etapa tem exercícios com **enunciado**, espaço para você resolver e a
**resposta comentada escondida** (clique em "Ver resposta" só depois de tentar).

> 💡 A regra de ouro: **tente resolver antes de abrir o gabarito.** O aprendizado
> de Go vem de escrever, errar e consertar — não de ler a solução pronta.

## ⚙️ Como usar

```bash
# 1. Clone o repositório
git clone https://github.com/SEU-USUARIO/learning-go.git
cd learning-go

# 2. Confirme que o Go está instalado
go version

# 3. Vá para uma etapa e crie seu arquivo de solução
cd 01-fundamentos
# escreva seu código em um arquivo .go e rode:
go run seu_arquivo.go
```

### Fluxo recomendado por exercício
1. Leia o enunciado.
2. Escreva sua solução num arquivo `.go`.
3. Rode e teste até funcionar.
4. **Só então** abra "Ver resposta" e compare com a sua.
5. Faça um commit: `git commit -m "etapa 01: exercicio 3 resolvido"`.
